很好，进入**第四步**。
这一步开始把“评价体系”真正接进**服务发现结果**里，让它开始产生可见效果。

## 第四步目标

做 4 件事：

1. 给 `Tool` 和 `Service` 返回结构增加可选 trust 摘要字段
2. 让 `/discovery/search` 和 `/client/discovery/search` 支持**兼容式增强参数**
3. 在 `SearchEngine.search()` 里加入**召回后轻量重排**
4. 让 `resolve / trending / featured / 服务详情` 也能带 trust 摘要

这一步做完后，你就能看到：

- 搜索结果里出现 `trust_score / success_rate / rating_count / avg_response_time_ms`
- 搜索顺序开始受到 trust 影响，但**匹配性仍然优先**

## 3.5 在 `SearchEngine` 里新增 3 个辅助方法

放在 `__init__()` 后面、`search()` 前面。

```python
    def _get_trust_stats_map(self, service_ids: List[int]) -> Dict[int, ServiceTrustStats]:
        """
        批量获取服务信任摘要映射
        
        Args:
            service_ids: 服务ID列表
        
        Returns:
            Dict[int, ServiceTrustStats]: service_id -> trust_stats 映射
        """
        if not service_ids:
            return {}

        trust_stats_list = self.db.query(ServiceTrustStats).filter(
            ServiceTrustStats.service_id.in_(service_ids)
        ).all()

        return {item.service_id: item for item in trust_stats_list}

    def _calc_cost_score(self, service: Service, max_price: Optional[float] = None) -> float:
        """
        计算价格分
        
        第一阶段使用轻量规则：
        - 免费服务得分较高
        - 在指定价格范围内，价格越低得分越高
        """
        price = service.price_per_unit or 0.0

        if price <= 0:
            return 100.0

        if max_price is not None and max_price > 0:
            ratio = min(price / max_price, 1.0)
            return round((1.0 - ratio) * 100.0, 2)

        # 未指定max_price时，使用简单分段
        if price <= 1:
            return 90.0
        if price <= 5:
            return 75.0
        if price <= 10:
            return 60.0
        return 40.0

    def _rerank_services(
        self,
        services: List[Service],
        match_scores: Optional[Dict[int, float]] = None,
        sort_by: str = "balanced",
        max_price: Optional[float] = None
    ) -> List[Service]:
        """
        对候选服务进行召回后轻量重排
        
        规则：
        - relevance：优先保持匹配性
        - trust：优先trust_score
        - balanced：匹配优先，再引入trust和价格因子
        """
        if not services:
            return services

        ranked = []

        for idx, service in enumerate(services):
            # 兼容无显式match_score场景
            default_match_score = max(0.0, 100.0 - idx * 5.0)
            match_score = match_scores.get(service.id, default_match_score) if match_scores else default_match_score

            trust_score = getattr(service, "trust_score", None)
            if trust_score is None:
                trust_score = 0.0

            success_rate = getattr(service, "success_rate", None)
            performance_score = success_rate if success_rate is not None else 0.0

            cost_score = self._calc_cost_score(service, max_price=max_price)

            if sort_by == "trust":
                final_rank = (
                    0.60 * trust_score +
                    0.20 * match_score +
                    0.10 * performance_score +
                    0.10 * cost_score
                )
            elif sort_by == "relevance":
                final_rank = (
                    0.80 * match_score +
                    0.10 * trust_score +
                    0.05 * performance_score +
                    0.05 * cost_score
                )
            else:
                # 默认balanced：匹配性优先，trust用于候选重排
                final_rank = (
                    0.55 * match_score +
                    0.25 * trust_score +
                    0.10 * performance_score +
                    0.10 * cost_score
                )

            ranked.append((final_rank, service))

        ranked.sort(key=lambda x: x[0], reverse=True)
        return [item[1] for item in ranked]
```
